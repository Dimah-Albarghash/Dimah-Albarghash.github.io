# Dimah Albarghash — Portfolio v3

A multi-page personal portfolio built with HTML, CSS, and vanilla JavaScript.

## Files

- `index.html` — Home
- `about.html` — About
- `education.html` — Education
- `projects.html` — Academic projects
- `certifications.html` — Certifications and training
- `skills.html` — Technical and soft skills
- `volunteering.html` — Leadership and volunteering
- `contact.html` — Contact page
- `style.css` — Responsive design and page themes
- `script.js` — Mobile menu, pointer effects, card interactions, and transitions

## Open locally

Open `index.html` in a browser.

## GitHub Pages

Upload all files to the root of the GitHub repository. The site is ready for GitHub Pages with no build process.

Make sure each Google Drive project uses **Anyone with the link → Viewer**.

## Visual identity update
The home-page name uses one unified Poppins font for the full name, with a subtle navy/blue animated sheen and pointer-following highlight. The site's original off-white, navy, muted blue, and gold palette is retained across all pages.
